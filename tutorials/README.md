## Notes / caveats

* The official tutorials are written mainly in C (and for Android in C/Java). ([gstreamer.freedesktop.org][1])
* The “final app source code” is typically distributed **with** GStreamer, not in a separate external GitHub repo — the tutorial pages refer to files like `basic-tutorial-3.c`, `playback-tutorial-4.c`, or Android project folders. ([gstreamer.freedesktop.org][11])
* For tutorials involving media (playback, streaming, video, audio), you’ll need appropriate media files or network-accessible media URIs (depending on the example).
* For Android tutorials you also need the Android SDK/NDK environment and GStreamer Android binaries.
